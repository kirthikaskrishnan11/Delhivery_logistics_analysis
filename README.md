# Delhivery Logistics Data Analysis

## Project Objective
Analyze logistics delivery data to identify operational inefficiencies and transportation patterns.

## Dataset
Delhivery logistics dataset containing trip-level and segment-level delivery information.

## Key Analysis
- Data Cleaning
- Feature Engineering
- Trip-level Aggregation
- Hypothesis Testing
- Corridor Analysis

## Tools Used
Python
Pandas
NumPy
Matplotlib
Seaborn
Scipy

## Key Insights
- Identified high-delay logistics corridors across different states.
- Compared actual delivery time with OSRM predicted time using hypothesis testing.
- Found significant differences between predicted and actual delivery metrics, indicating potential inefficiencies in route estimation.
